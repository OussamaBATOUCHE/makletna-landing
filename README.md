# Makletna — Landing Page

Static landing page for [Makletna](https://makletna.com) — an Algeria-first app connecting users with home-made food from local cooks nearby.

## Tech Stack

- React 18 + TypeScript
- Vite
- Framer Motion
- Trilingual: English, Arabic (RTL), French

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The output will be in the `dist/` folder — deploy anywhere (Vercel, Netlify, GitHub Pages, etc.).
